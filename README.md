# Overzicht giften aan politieke partijen voor de Tweede Kamer verkiezingen
Op 22 oktober 2025 werd een overzicht van giften aan politieke partijen die groter waren dan € 1000 gepubliceerd.
Dit is een speciale publicatie in verband met de Tweede Kamerverkiezingen op 29 oktober.

Hier maken we inzichtelijk welke updates er in het overzicht gemaakt zijn. Dit project is een
aangepaste versie van `Overzicht substantiële giften aan politieke partijen` (`https://github.com/openstate/ro-pp-subsidies.git`)

# Running
- Copy `update.sh.example` to `update.sh` and fill in your GitHub token
- `cd docker && docker-compose up -d`
- Add a cronjob for `update.sh`
